# memdb

---

## About

"Concurrency in Go" course project.

## Build

Run in terminal:

```shell
make build
```

## Running tests

Run in terminal:

```shell
make test
```

## Run application locally

Rename `example.memdb.conf.yml` to `memdb.conf.yml`.

Change configuration values to preferred settings.

Run memdb:

```shell
make run-memdb
```

And then run client:

```shell
make run-client
```
